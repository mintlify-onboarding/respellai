#### API Block

The API block is our most powerful non-model block. It allows you to call any API endpoint using the REST architecture. You can encrypt API keys using the Secrets section, add a URL, add headers, and select an API method when using this block.

![](/images/edcc1e4-image.png)

#### File to Text Block

The file to text block converts any PDF file or image to a strings of text transcribing its content.

![](/images/325ca77-image.png) 

#### Web Search Block

The web search block makes a live search engine query based on the input provided and returns the first page of results.

![](/images/f2a528b-image.png)
